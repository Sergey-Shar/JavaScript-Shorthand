# JavaScript-Shorthand

1. The Ternary Operator 

Longhand:

```javascript
const x = 20;
let answer;

if (x > 10) {
    answer = "greater than 10";
} else {
    answer =  "less than 10";
}  console.log"less than 10";
}
```
Shorthand:

```javascript
const answer = x > 10 ? "greater than 10" : "less than 10";
```
